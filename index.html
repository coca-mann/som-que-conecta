Projeto Concluído

A plataforma "Som que Conecta" cumpriu seu propósito.

Esta aplicação foi desenvolvida como meu Trabalho de Conclusão de Curso (TCC), com o objetivo de demonstrar a criação de uma plataforma web completa para educação musical, conectando alunos, professores e instrumentos.

O servidor que hospedava a aplicação dinâmica foi desativado, mas o espírito do projeto continua vivo!

Você pode explorar todo o código-fonte, a arquitetura do projeto e a documentação no repositório oficial no GitHub.
